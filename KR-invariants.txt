Aside on drawing knots
======================

Use

load 31mar04a.k    % knot to load. Use Rolfsen's notation (e.g. 6.1)
psmode=40 % the style
pserase=1.8 % the amount of whitespace to erase around crossings
 
psout 3compBrunB % output to ps file

In the ps file the actual drawing is done by lines like

434.75 319.94 437.72 327.20 441.36 333.72 446.02 340.17 P0
430.33 300.05 430.54 306.74 432.05 313.37 435.03 320.63 P0

each such line produces a small arc segment, painted in the "style" P0

PS guide: http://www.tailrecursive.org/postscript/postscript.html
