Hopf link (aka L2a1, two components, see http://katlas.org/wiki/Image:L2a1.gif)

(t2q2+t2+q6+q4)/(t2)
Agrees with Gukov-Iqbal-Kozcaz-Vafa 07 (GIKV), p.15.

(t2q4+t2q2+t2+q10+2q8+2q6+q4)/(t2)

N = 4:
(t2q6+t2q4+t2q2+t2+q14+2q12+3q10+3q8+2q6+q4)/(t2)

Trefoil (aka 3_1, see http://katlas.org/wiki/Image:3_1.gif)

N = 2:
(q9+q5t+q3t3+qt3)/(t3)
Agrees with the data on the Knot Atlas.

N = 3:
(q14+q12+q8t+q6t3+q6t+q4t3+q2t3)/(t3)
Agrees with Gukov-Schwarz-Vafa (GSV), eq. (4.8).

N = 4:
(q19+q17+q15+q11t+q9t3+q9t+q7t3+q7t+q5t3+q3t3)/(t3) 
Took 103 seconds.
Agrees with GSV, eq. (4.9).

N = 5:
(q24+q22+q20+q18+q14t+q12t3+q12t+q10t3+q10t+q8t3+q8t+q6t3+q4t3)/(t3)
Took 176 seconds.
Agrees with GSV, eq. (4.10).

N = 6:
(t3q15+t3q13+t3q11+t3q9+t3q7+t3q5+tq17+tq15+tq13+tq11+tq9+q29+q27+q25+q23+q21)/(t3)
State 8 took 3325 seconds.

Figure-eight knot (aka 4_1, see http://katlas.org/wiki/Image:4_1.gif)

N = 2:
(q10+q6t2+q6t+q4t3+q4t2+t4)/(q5t2)

(q16+q14+q10t2+q10t+q8t3+q8t2+q8t+q6t3+q6t2+q2t4+t4)/(q8t2)

Agrees with Gukov-Dunfield-Rasmussen eq. (64) after q <--> 1/q.
